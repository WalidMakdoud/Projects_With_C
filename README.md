# Projects_With_C

Une collection de projets en **langage C**, avec des programes de jeux et applications administratives.

---

## Contenu

Voici les fichiers et dossiers principaux :

### 1. [Projets Administratifs](./Projets%20Administratifs)
Ce dossier contient les projets **administratifs** en C, comme la gestion des stocks.  

**Les fichiers :**
- [GestionDeStock.c](./Projets%20Administratifs/GestionDeStock.c) : permet d’ajouter, supprimer ou modifier des produit dans un stock.  

---

### 2. [Projets Des Jeux](./Projets%20Des%20Jeux)
Ce dossier contient des projets **de jeux** réalisés en C. Il est divisé en deux sous-dossiers :  

#### a) [Version 1](./Projets%20Des%20Jeux/Version%201)
Contient les programmes séparés pour chaque jeu :  
- [Jeu_Du_Nombre_Mystere.c](./Projets%20Des%20Jeux/Version%201/Jeu_Du_Nombre_Mystere.c) : deviner le nombre mystère choisi aléatoirement.  
- [Jeu_Du_Pendu.c](./Projets%20Des%20Jeux/Version%201/Jeu_Du_Pendu.c) : le jeu du pendu pour un mot à deviner.  
- [Pierre_Feuille_Ciseaux.c](./Projets%20Des%20Jeux/Version%201/Pierre_Feuille_Ciseaux.c) : jeu pierre-feuille-ciseaux contre l’ordinateur.  
- [Tic_Tac_Toe.c](./Projets%20Des%20Jeux/Version%201/Tic_Tac_Toe.c) : morpion (X | O) pour deux joueurs.  

#### b) [Version Finale](./Projets%20Des%20Jeux/Version%20Finale/)
Contient **tous les jeux combinés dans un seul programme ******"WalTendo"******** :  
- [WalTendo.c](./Projets%20Des%20Jeux/Version%20Finale/WalTendo.c) : permet de choisir et jouer à tous les jeux (Nombre Mystère, Pendu, Pierre-Feuille-Ciseaux, Tic-Tac-Toe) depuis un menu.
